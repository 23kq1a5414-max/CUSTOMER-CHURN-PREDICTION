# 📊 Customer Churn Analyzer — Sophisticated Dark Edition

## Overview
The **Customer Churn Analyzer** is a full-stack application designed to help businesses predict and mitigate customer churn. Built with **100% type safety** and styled in a **Sophisticated Dark theme**, it combines advanced machine learning capabilities with an elegant, professional interface.

---

## 🎨 Visual Identity
- **Ambient Noir Palette**: Dark off-black canvas (#0A0A0A) with subtle grid lines (#2A2A2A) and gold highlights (#C5B48C).
- **Typography**: Serif italic headers paired with responsive caps for clarity.
- **Micro-interactions**: Smooth hover states, active navigation markers, and responsive feedback.

---

## ⚡ Key Features
### Model Simulator Workspace
- Compare algorithms: XGBoost, Random Forests, Decision Trees, Logistic Regression.
- Real-time hyperparameter tuning (learning rates, forest tree counts, max split depth, L1/L2 penalties).
- Interactive SVG graphs: ROC curves, validation loss trends, correlation matrices.
- Dynamic confusion matrix across Telecom, Fintech, and SaaS datasets.

### Client-Side CSV ML Engine
- Drag-and-drop CSV upload.
- Automatic dataset profiling (numerical vs categorical, missing values).
- In-browser Logistic Regression training with SGD.
- Live training loss curve visualization.

### Individual Predictive Calculator
- Real-time churn probability gauge.
- Risk classification (Low, Medium, High).
- Prescriptive retention strategies tailored to risk levels.

### AI Python ML Expert
- Conversational consultant powered by Gemini 3.5 Flash.
- Quick-access template cards for preprocessing, SMOTE, and hyperparameter tuning.
- Generates ready-to-use Python ML code snippets.

## 🛠 Tech Stack
- **Frontend**: Responsive UI with custom SVG visualizations.
- **Backend**: Secure Express server proxy.
- **ML Integration**: Gemini 3.5 Flash + client-side Logistic Regression.
- **Design**: Sophisticated Dark theme with gold highlights.
